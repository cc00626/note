### px

逻辑像素单位



### em

参考父元素的 font- size 



### rem

**根元素 (`<html>`)** 的 `font-size`



### vw / vh

根据浏览器的窗口大小计算



### %

参考父元素

**盒子模型相关：参考宽度**

- width
- padding
- margin

**定位与宽高相关**

- height **参考父元素高度**
- top / bottom **参考父元素高度**
- left / right **参考父元素的宽度**

**文本与行高相关**

- **`font-size`**：参考 **父元素** 的 `font-size`。

- **`line-height`**：参考 **当前元素自身** 的 `font-size`。

**变换与偏移：参考“自己”**

- **`transform: translate(-50%, -50%)`**：参考 **该元素自身** 的宽度和高度。

- **区别**：`left: 50%` 是把盒子的左侧移动到父容器的中间；而 `translate(-50%)` 是把盒子往回挪动自身宽度的一半，从而实现视觉上的绝对居中。
