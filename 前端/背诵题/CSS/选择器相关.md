### css优先级

| **等级**   | **代表的对象**                                          | **权重值 (示意)** |
| ---------- | ------------------------------------------------------- | ----------------- |
| **第一级** | **内联样式** (Inline Styles)                            | `(1, 0, 0, 0)`    |
| **第二级** | **ID 选择器** (`#id`)                                   | `(0, 1, 0, 0)`    |
| **第三级** | **类、伪类、属性选择器** (`.class`, `:hover`, `[type]`) | `(0, 0, 1, 0)`    |
| **第四级** | **元素、伪元素选择器** (`div`, `::before`)              | `(0, 0, 0, 1)`    |

### 伪类选择器和伪元素选择器的区别

在于是否创建了内容

**伪类选择器：**

相当于添加了类名

可以叠加使用

**伪元素选择器：**

相当于添加了一个标签

一个选择器中只能选择一个
