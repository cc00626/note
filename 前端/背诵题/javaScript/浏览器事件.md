## DOMContentLoaded和window.onload

| **特性**       | **DOMContentLoaded**                        | **window.onload**                                  |
| -------------- | ------------------------------------------- | -------------------------------------------------- |
| **触发时机**   | **HTML 文档被完全加载和解析**完成后触发。   | **整个页面所有资源**加载完成后触发。               |
| **等待资源**   | 不等待 CSS、图片、iframe 等外部资源。       | **等待**所有外部资源（图片、样式、脚本、iframe）。 |
| **性能意义**   | 代表 **DOM 树构建完毕**，此时可以操作节点。 | 代表 **页面完全可用**。                            |
| **执行优先级** | **先执行**                                  | **后执行**                                         |
