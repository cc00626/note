### 修改数组的方法

| **方法**        | **描述**                               | **返回值**                 |
| --------------- | -------------------------------------- | -------------------------- |
| **`push()`**    | 在数组**末尾**添加一个或多个元素       | 改变后的数组**长度**       |
| **`pop()`**     | 删除数组**最后一个**元素               | 被删除的那个**元素**       |
| **`unshift()`** | 在数组**开头**添加一个或多个元素       | 改变后的数组**长度**       |
| **`shift()`**   | 删除数组**第一个**元素                 | 被删除的那个**元素**       |
| **`splice()`**  | **万能方法**：可以删除、插入、替换元素 | 包含被删除元素的**新数组** |
| **`sort()`**    | 对数组进行排序（默认按 Unicode）       | 排序后的**原数组**         |
| **`reverse()`** | 反转数组中元素的顺序                   | 反转后的**原数组**         |



### 不会修改数组的方法

**`concat()`**: 合并两个或多个数组。

**`slice(start, end)`**: 浅拷贝数组的一部分。注意是“左闭右开”，即包含 `start` 但不包含 `end`。

**`join(separator)`**: 将数组所有元素连接成字符串。

**`flat()`**: 扁平化嵌套数组。你可以指定深度 `arr.flat(2)` 或者使用 `Infinity`。



### 数组遍历的方法

`map()` vs `forEach()`

- **`map()`**: 创建一个新数组，结果是原数组每个元素调用函数后的返回值。**必须有 return。**
- **`forEach()`**: 只是单纯的循环，没有返回值（返回 `undefined`）。通常用于触发副作用（如打印日志、修改外部变量）。

`filter()`

- 过滤出符合条件的元素。返回一个满足条件的新数组。如果没有任何元素符合，则返回空数组 `[]`。

`reduce()` (大杀器)

- 用于累加、归并。
- **语法**: `arr.reduce((accumulator, currentValue) => { ... }, initialValue)`
- **场景**: 求和、数组转对象、统计元素出现次数等。



### 查找和校验

**`find()`**: 返回**第一个**符合条件的元素。找不到则返回 `undefined`。

**`findIndex()`**: 返回**第一个**符合条件的元素的索引。找不到则返回 `-1`。

**`includes()`**: 判断数组是否包含某个值，返回 `true/false`。相比 `indexOf`，它能识别 `NaN`。

**`some()`**: 只要有一个元素满足条件，就返回 `true`。

**`every()`**: 必须所有元素都满足条件，才返回 `true`。
