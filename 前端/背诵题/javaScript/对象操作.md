属性遍历与转换

| **方法**                   | **功能**                       | **返回值**                   |
| -------------------------- | ------------------------------ | ---------------------------- |
| **`Object.keys()`**        | 获取对象所有可枚举的**键**     | 字符串数组                   |
| **`Object.values()`**      | 获取对象所有可枚举的**值**     | 任意类型数组                 |
| **`Object.entries()`**     | 获取对象所有可枚举的**键值对** | 二维数组 `[[key, val], ...]` |
| **`Object.fromEntries()`** | 将键值对列表转回对象           | 一个新对象                   |



对象的合并和拷贝

**`Object.assign(target, ...sources)`**：将源对象的所有**可枚举自有属性**复制到目标对象。

- **注意**：这是**浅拷贝**。
- **注意**：如果有同名属性，后面的会覆盖前面的。

**展开运算符 `{...obj}`**：虽然不是方法，但在现代开发中几乎取代了 `assign`。



原型链管理

**`Object.create(proto, [descriptors])`**：

- 以 `proto` 对象为原型创建一个新对象。
- **妙用**：`Object.create(null)` 可以创建一个**没有任何原型属性**（连 `toString` 都没有）的绝对空对象，常用于实现高性能的哈希映射（Dictionary）。

**`Object.getPrototypeOf(obj)`**：获取对象的原型。比过时的 `obj.__proto__` 更规范。

**`Object.setPrototypeOf(obj, proto)`**：设置对象的原型（性能开销大，生产环境慎用）。



属性描述符和响应式

**`Object.defineProperty(obj, prop, descriptor)`**：

- 精确添加或修改属性。
- **Descriptor 关键配置**：`value`, `writable` (是否可写), `enumerable` (是否可遍历), `configurable` (是否可删除/修改配置), 以及 `get/set`（访问器）。

**`Object.defineProperties(obj, props)`**：一次定义多个属性。

**`Object.getOwnPropertyDescriptor(obj, prop)`**：获取指定属性的详细描述符。



对象状态的锁定

**`Object.preventExtensions(obj)`**：**禁止添加**新属性，但可以删除和修改旧属性。

**`Object.seal(obj)`**：**密封**。禁止添加、删除属性，但**可以修改**现有值。

**`Object.freeze(obj)`**：**冻结**。禁止添加、删除、修改任何属性（最高等级）



逻辑判断和比较

**`Object.is(val1, val2)`**：

- 判断两个值是否完全相等。
- **对比 `===` 的优势**：它能正确识别 `Object.is(NaN, NaN) // true` 以及 `Object.is(+0, -0) // false`。

**`obj.hasOwnProperty(prop)`**：

- 判断属性是否为对象的**自有属性**，而不是原型链上的属性。
- **现代写法**：`Object.hasOwn(obj, prop)`（ES2022 引入，更安全，防止 `obj` 没原型）。
