### useEffect的闭包问题

react每次执行都会重新执行

```
function App() {
  const [count, setCount] = useState(0)

  function handleClick() {
    setTimeout(() => {
      console.log(count)
    }, 1000)
  }

  return <button onClick={handleClick}>点</button>
}
```



```
useEffect(() => {
  setInterval(() => {
    console.log(count)
  }, 1000)
}, [])
```



解决

```
const countRef = useRef(count)

useEffect(() => {
  countRef.current = count
})

function handleClick() {
  setTimeout(() => {
    console.log(countRef.current)
  }, 1000)
}
```
