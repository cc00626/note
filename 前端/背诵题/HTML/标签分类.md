### 行内元素，块级元素，行内块元素

**行内元素**  

1. 不会自动换行，一行撑满才进行换行
2. 不可以设置宽高属性 ，内容由自身的高度撑开。
3. 设置左右 margin 和 padding 均有效；设置上下 margin 完全无效，设置上下 padding 会在视觉上生效（如背景色变高），但不占据空间（不影响文档流排版，会与其他元素重叠）。
4. `span`、`a`、`i`、`em`、`strong`

**块级元素**

1. 会自动进行换行，不设置宽度的情况下，默认是100%
2. 可以设置宽度和高度
3. 可以设置四个方向的padding和margin
4. `div`、`p`、`h1-h6`、`ul`、`li`、`form`

**行内块元素**

1. 不可以自动换行
2. 可以设置宽度和高度
3. 可以设置四个方向的padding和margin，默认排序是从左到右
4. 常见标签（包含天然具有此类特性的替换元素）：`img`、`textarea`、表单控件 `button` 以及 `input`。（注：你之前提到的 `table` 和 `td` 在视觉表现上虽有相似之处，但在 CSS 规范中默认分别属于特定的 `table` 和 `table-cell` 类型）



### 替换元素，非替换元素

**替换元素**

内容由外部的资源或者标签的属性来定义内容

**非替换元素**

元素的内容直接由标签所定义
