### 使用 `<component :is="...">`

这是动态组件的灵魂。`:is` 绑定的值可以是**组件的名称**（字符串）或者**组件的定义**（对象）

```
<template>
  <div class="tabs">
    <button @click="currentTab = 'Home'">首页</button>
    <button @click="currentTab = 'Article'">文章</button>
    <button @click="currentTab = 'User'">个人中心</button>

    <hr />

    <component :is="tabs[currentTab]" />
  </div>
</template>

<script setup>
import { ref } from 'vue';
import Home from './Home.vue';
import Article from './Article.vue';
import User from './User.vue';

// 维护一个组件映射表
const tabs = {
  Home,
  Article,
  User
};

const currentTab = ref('Home');
</script>
```



### 配合 `<KeepAlive>`

默认情况下，当动态组件切换时，旧组件会被**销毁**，新组件会被**创建**。这意味着你在“首页”输入的数据，切换到“文章”再回来时就没了。

如果你想保留组件状态，需要用 `<KeepAlive>` 包裹：

```
<KeepAlive>
  <component :is="tabs[currentTab]" />
</KeepAlive>
```
