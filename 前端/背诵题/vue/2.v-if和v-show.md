### 区别

| **特性**              | **v-if**                              | **v-show**                               |
| --------------------- | ------------------------------------- | ---------------------------------------- |
| **渲染本质**          | **动态添加/删除** DOM 元素            | 仅切换 CSS 的 `display: none`            |
| **编译过程**          | 惰性执行。初始为假时，完全不渲染      | 无论初始真假，都会编译并保留在 DOM 中    |
| **生命周期**          | 会触发组件的生命周期钩子（销毁/挂载） | 不会触发生命周期钩子                     |
| **性能消耗**          | **切换开销高**（频繁增删 DOM）        | **初始渲染开销高**（无论是否显示都渲染） |
| **适用场景**          | 运行后很少改变状态的场景              | 需要**频繁切换**显示的场景               |
| **配合 `<template>`** | 支持，可以包裹多个元素                | **不支持**（因为 template 没有物理节点） |
