| **生命周期钩子 (Vue 3)** | **执行时机**                              | **常见用途**                                                 |
| ------------------------ | ----------------------------------------- | ------------------------------------------------------------ |
| **`setup()`**            | 组件实例创建前执行，此时 `props` 已解析。 | 代替了 Vue 2 的 `beforeCreate` 和 `created`。用于初始化响应式数据、定义函数。 |
| **`onBeforeMount`**      | 组件挂载到 DOM 之前。                     | 此时模板已编译成渲染函数，但页面还没看到 DOM。               |
| **`onMounted`**          | 组件已经渲染完成，DOM 节点已生成。        | **核心钩子**。发送网络请求、操作 DOM、启动定时器、绑定第三方库（如 Echarts）。 |



| **生命周期钩子 (Vue 3)** | **执行时机**                        | **常见用途**                                     |
| ------------------------ | ----------------------------------- | ------------------------------------------------ |
| **`onBeforeUpdate`**     | 数据已变，但 DOM 还没更新。         | 可以在此处访问更新前的 DOM 状态。                |
| **`onUpdated`**          | 数据变了，且 DOM 已经重新渲染完毕。 | 避免在此处直接修改数据，否则可能导致**死循环**。 |



| **生命周期钩子 (Vue 3)** | **执行时机**   | **常见用途**                                                 |
| ------------------------ | -------------- | ------------------------------------------------------------ |
| **`onBeforeUnmount`**    | 组件销毁之前。 | **清理现场**。清除定时器、取消订阅、解绑全局 DOM 事件（如 `window.addEventListener`）。 |
| **`onUnmounted`**        | 组件完全销毁。 | 此时组件的所有指令已解绑，子组件也已销毁。                   |



**`onActivated` / `onDeactivated`**: 配合 `<keep-alive>` 使用。当组件被缓存时，它不会被销毁，而是进入“失活”状态。这两个钩子负责处理切入和切出。

**`onErrorCaptured`**: 后代组件发生错误时被调用。可以用它来做前端错误日志上报。
