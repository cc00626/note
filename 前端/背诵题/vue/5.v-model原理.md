### Vue 2.x 的实现

```
<Child v-model="searchText" />

<Child :value="searchText" @input="searchText = $event" />
```



### Vue 3.x 的实现

```
<Child v-model="searchText" />

<Child :modelValue="searchText" @update:modelValue="searchText = $event" />
```



在 Vue 2 中，`v-model` 是“霸道”的，它默认占用了组件的 `value` 属性和 `input` 事件。如果你想在一个表单组件里同时双向绑定“用户名”、“密码”和“验证码”，Vue 2 处理起来非常优雅度不足。

**Vue 3 的改进：** 通过将默认值改为 `modelValue` 并支持 `v-model:xxx` 语法，你可以极其轻松地在同一个组件上挂载无限个双向绑定。
